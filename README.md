A React 19 + TypeScript elevator simulation system.

## Features

- Dynamic floor and elevator configuration.
- Closest elevator assignment algorithm.
- Call queue handling.
- Smooth CSS animations.
- Arrival sound effects.

### Installation

```bash
npm install
```

### Run the App

```bash
npm run dev
```

### Run Tests

```bash
npm test
```

## Tech Stack

- React 19
- TypeScript
- Vite
- Vitest
- React Icons
